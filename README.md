# Business Lending Portfolio Analysis

A Power BI dashboard analysing a **$302.23M** business lending portfolio of **2,689 loans** across customer types, industries, and US states — built to surface portfolio concentration risk, settlement performance, and data quality gaps.

**Status:** 🚧 In Progress

## Dashboard Pages

1. **Executive Overview** — portfolio-level KPIs and high-level performance snapshot
2. **Category Data Quality** — completeness and classification issues across loan categories
3. **Portfolio Health** — concentration, risk, and settlement metrics

## Key Insights

- Top-financed category sits at **5.46% concentration** (**$16.51M**), flagged for concentration risk monitoring
- Unsettled rates range from **23.30%** (Manufacturing) to **34.67%** (Professional, Scientific and Technical Services)
- **$29.95M** in unclassified exposure identified, requiring reclassification

## Tools & Methods

- **Power BI** — report build and visualisation
- **Power Query (M)** — data shaping and modelling
- **DAX** — measure authoring (concentration ratios, settlement rate calculations, category-level aggregations)
- **Claude** — used to validate and optimise DAX measure logic

## Data

The dataset is a synthetic business lending portfolio (2,689 loan records) covering customer type, industry classification, loan amount, US state, and settlement status. It does not represent real customer or institutional data.
